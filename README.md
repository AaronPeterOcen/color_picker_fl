# color_picker_fl

A Flutter application for generating and managing color palettes. This app allows you to create, customize, and save color palettes with ease.

## Features

- **Random Color Generation**: Generate random color palettes with a single tap
- **Color Locking**: Lock specific colors to keep them while regenerating others
- **Palette Management**: Save and manage your favorite color palettes
- **Hex Color Display**: View hex codes for each color in the palette
- **Multi-platform Support**: Runs on iOS, Android, macOS, Windows, Linux, and Web

## How to Use

1. **Generate a Palette**: Open the app to see a randomly generated color palette
2. **Lock Colors**: Tap on any color to lock it in place
3. **Regenerate**: Generate new colors while keeping your locked colors intact
4. **View Hex Codes**: Copy hex color codes for use in your design projects
5. **Save Palettes**: Save your favorite palettes with custom names for future reference

## Installation

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (version 3.0 or higher)
- [Dart SDK](https://dart.dev/get-dart) (comes with Flutter)
- A code editor (VS Code, Android Studio, or IntelliJ IDEA)

### Steps

1. **Clone the repository**:

   ```bash
   git clone https://github.com/yourusername/color_picker_fl.git
   cd color_picker_fl
   ```

2. **Install dependencies**:

   ```bash
   flutter pub get
   ```

3. **Run the app**:

   ```bash
   flutter run
   ```

4. **Build for production** (optional):
   ```bash
   flutter build apk        # For Android
   flutter build ios        # For iOS
   flutter build web        # For Web
   ```
